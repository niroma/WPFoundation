== Description ==

WPFoundation is a free to use HTML5 WordPress Theme based on Zurb's Foundation framework (https://foundation.zurb.com) and TidyThemes' BlankSlate Theme (https://fr.wordpress.org/themes/blankslate/). WPFoundation also comes with Font Awesome Free Icon pack !

If you're creating your own theme or client project, open up all files and do a "Find and Replace All in All Files" on the word "wpfoundation" with your own slug and prefix.
